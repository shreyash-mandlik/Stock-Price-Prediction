# Stock Price Prediction 📈

LSTM Neural Network that predicts future stock prices 
for major companies like Apple, Google, Tesla and more.

## 🌐 Live Demo
👉 [Try the app here!](https://stock-price-prediction-a8mdv2btjim3r25rnjozax.streamlit.app/)

## 🎯 Features
- Select from 6 major stocks (AAPL, GOOGL, MSFT, AMZN, TSLA, META)
- Custom date range selection
- 7-60 day price forecast
- Interactive charts
- Real time stock data via Yahoo Finance

## 🛠️ Tech Stack
- Python, PyTorch (LSTM)
- yfinance (real time data)
- Scikit-learn (MinMaxScaler)
- Streamlit, Matplotlib

## 📊 Model Architecture
- 2 layer LSTM with 50 hidden units
- Dropout 0.2 to prevent overfitting
- Trained on 60 day sequences
- MAE: ~$7-9 on Apple stock

## 📈 Results
- Apple: $33 → $190 (480% growth 2019-2023)
- Google: $50 → $138 (172% growth 2019-2023)
